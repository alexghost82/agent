FrontendReact 19, Vite, TypeScript, Tailwind CSS, Zustand

Computer Vision UIMediaPipe face detection/blur, segmentation overlay

Операционный смысл

Frontend — глаза оператора. Backend — память и мозг. YOLO — быстрый детектор объектов. Vision model — наблюдатель, который объясняет сцену человеческим языком.
